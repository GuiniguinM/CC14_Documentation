Introduction

Bellow are the lessons I learned during the journey in CC14.

GitHub

Importance:

GitHub is a platform for version control and collaboration. It's widely used in software development for managing codebases, tracking changes, and facilitating teamwork.

Key Notes:

  • Repositories: Projects are organized into repositories where code, issues, and project documentation are stored.
  • Branches: Developers work on separate branches to isolate changes before merging them into the main codebase (often called the "master" branch).
  • Pull Requests: Changes are proposed and reviewed through pull requests, allowing for discussion and feedback before merging.

Shortcuts:

  • git clone (repository url): Clone a repository to your local machine.
  • git add .: Add all changes in the current directory to the staging area.
  • git commit -m "Commit message": Commit staged changes with a descriptive message.
  • git push origin (branch name): Push committed changes to a remote repository.
  • git pull: Fetch and merge changes from the remote repository.
VS Code (Visual Studio Code)

Importance:

VS Code is a popular code editor known for its versatility, extensive features, and customization options.

Key Notes:

  • Extensions: VS Code supports a vast array of extensions for language support, debugging, version control, and more.
  • Integrated Terminal: It includes a built-in terminal, allowing developers to run commands without leaving the editor.
  • IntelliSense: Provides code completion suggestions, syntax highlighting, and other intelligent features based on context.

Shortcuts:

  • Ctrl + (Backtick): Open integrated terminal.
  • Ctrl + Shift + P: Open command palette for quick access to various commands.
  • Ctrl + Shift + L: Select all occurrences of the current selection.
  • Ctrl + /: Toggle line comment.
  • Ctrl + Space: Trigger IntelliSense
Basic HTML

Importance:

HTML (Hypertext Markup Language) is a fundamental languages for building web pages.

Key Notes:

  • HTML: Defines the structure and content of web pages.

Shortcuts:

  • < !DOCTYPE html>: Declaration of HTML5.
  • < tagname>: Opening tag of an HTML element.
  • < /tagname>: Closing tag of an HTML element.
CSS

Importance:

CSS (Cascading Style Sheets) is used to better enhance the styles in your html file.

Key Notes:

  • CSS: Controls the presentation and layout, including styles like colors, fonts, and positioning.

Shortcuts:

  • selector { property: value; }: Basic CSS syntax.
  • color: #hex_value;: Set text color.
  • font-family: font_name;: Specify font family.
  • margin: value;, padding: value;: Set margin and padding.
WSL

Importance:

WSL (Windows Subsystem for Linux) allows running a Linux distribution alongside Windows, enabling developers to use Linux tools and workflows natively on Windows.

Key Notes:

  • Integration: Provides seamless integration between Windows and Linux environments, including file system access and command-line tools.
  • Compatibility: Supports a wide range of Linux distributions, such as Ubuntu, Debian, and CentOS.
  • Development: Enables running Linux-specific development environments and tools without needing a separate Linux machine or virtualization.

Shortcuts:

  • wsl: Launches the default Linux distribution.
  • wsl --list: Lists all installed Linux distributions.
  • wsl --set-version : Sets the WSL version for a distribution.
  • wsl --shutdown: Shuts down all running WSL distributions.
Linux

Importance:

Linux is an open-source operating system widely used in server environments, development, and embedded systems.

Key Notes:

  • Distributions: There are various Linux distributions (distros) tailored for different purposes, such as Ubuntu, Fedora, and CentOS.
  • Terminal: The command-line interface (CLI) is a central part of Linux, providing powerful tools and utilities for system management and development.
  • Package Management: Linux distributions use package managers like apt (for Debian-based) and yum (for Red Hat-based) to install, update, and manage software packages.

Shortcuts:

  • Ctrl + C: Interrupt current process.
  • Ctrl + D: Logout or exit terminal.
  • Ctrl + Z: Suspend a process.
  • Ctrl + R: Search command history.
  • Ctrl + L: Clear the terminal screen.
